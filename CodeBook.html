Data for tidy dataset is taken from Human Activity Recognition Using Smartphones Dataset. This dataset was filtered by non-mean and non-std data. In tidy dataset all filtered data was grouped by userId and taking action, so it contain 144 row for 24 people and 6 actions. All data is normalized to [-1, 1] interval.
